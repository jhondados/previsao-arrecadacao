# 🏛️ Previsão de Arrecadação

> Previsão de arrecadação tributária com séries temporais e fatores macroeconômicos. Apoia planejamento orçamentário e política fiscal.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/previsao-arrecadacao/

## 📊 Resultados-chave (KPIs)
**2.4%** MAPE · **12 meses** Horizonte · **15** Tributos · **96%** Acerto

## 🧠 Tecnologias de IA
Time Series · ML · Econometrics

## ✨ O que faz
- Previsão por tributo
- Fatores macro (PIB, inflação)
- Cenários fiscais
- Detecção de desvios

## 🗄️ Fonte de dados real
[Receita Federal + IBGE](https://www.gov.br/receitafederal)

## ⚙️ Como funciona
1. **Coleta de dados reais** — integração com Receita Federal + IBGE
2. **Processamento com IA** — Time Series · ML · Econometrics
3. **Dashboard & insights** — KPIs, gráficos e recomendações acionáveis
4. **Ação automatizada** — alertas, predições e decisões em tempo real
